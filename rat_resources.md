## mRatBN7.2

### the  new rat reference genome 

a collboration between Darwin Tree of Life Project, Wellcome Sanger Institute, the Rat Genome Database, and others

* [reference geneme sequence data from NCBI](https://www.ncbi.nlm.nih.gov/assembly/GCF_015227675.2/)
* [gene annotation from NCBI (AR108)](https://www.ncbi.nlm.nih.gov/genome/annotation_euk/Rattus_norvegicus/108/)
	* [announcement](https://ncbiinsights.ncbi.nlm.nih.gov/2021/02/12/rat-mratbn7-2-annotation/)
	* [ncbi data directory](https://ftp.ncbi.nlm.nih.gov/genomes/all/GCF/015/227/675/GCF_015227675.2_mRatBN7.2/)
	* [gtf file]( https://ftp.ncbi.nlm.nih.gov/genomes/all/GCF/015/227/675/GCF_015227675.2_mRatBN7.2/GCF_015227675.2_mRatBN7.2_genomic.gtf.gz)
	* gene annotation from Ensembl (pending)
* genome browser	
	* [NCBI browser](https://www.ncbi.nlm.nih.gov/genome/gdv/browser/genome/?id=GCF_015227675.2)
	* UCSD (pending)
	* Ensembl (pending)
* [raw genome sequencing data](https://vgp.github.io/genomeark/Rattus_norvegicus/)
* [rat genome joins the reference genome consortium](http://genomeref.blogspot.com/2020/11/a-new-rat-genome-assembly-sparks.html)
