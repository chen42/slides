# Long term Behavioral experiments with complicated pipelines 

* Many animals, will need unique IDs
* Behavioral pipeline has different stages 
* Need to track where each animal is in the pipeline
* Moving to the next stage may depend on meeting certain criteria  
* Raw data need to be coverted into a more convenient format
* Animals run on the same day likely to be on different stages 
* Breeding further increases complexity of the tracking need 
* There are many relevant pieces of information with unknown usefulness 
	* solution expiration dates
	* animal specific information (e.g. unexpected death)
